# OkulYokla — Dijital Yoklama Sistemi

Okul yoklama ve devamsızlık takibini kolaylaştıran, Supabase tabanlı dijital yoklama uygulaması.

## 🚀 GitHub Pages ile Kullanım

Bu uygulama GitHub Pages üzerinden PWA olarak çalışır.  
Yayınlandıktan sonra telefonda **"Ana ekrana ekle"** diyerek uygulama gibi kullanabilirsiniz.

## ⚙️ Kurulum

1. Supabase projesindeki URL ve Anon Key bilgilerini uygulamaya girin
2. Veritabanı tablolarını oluşturun
3. Giriş yapın ve kullanmaya başlayın

## 📱 PWA Olarak Yükleme

- **Android (Chrome):** Tarayıcı menüsü → "Ana ekrana ekle"
- **iPhone (Safari):** Paylaş → "Ana Ekrana Ekle"
